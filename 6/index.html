🔤 동사의 3가지 종류
그룹특징예시 동사
1그룹 ① 어미가 る로 끝나지 않는 동사
② 어미가 る로 끝나지만 る 앞이 あ·う·お단인 동사 		会あう / あそぶ / 行いく / すわる / はなす / よむ
2그룹 어미가 る로 끝나고 る 앞이 い단·え단인 동사 見みる / 食たべる
3그룹 불규칙 활용 동사 (딱 2개) 来くる（くる） / する
💡 見みる → る 앞이 「み (い단)」→ 2그룹    食たべる → る 앞이 「べ (え단)」→ 2그룹
💡 すわる → る 앞이 「わ (あ단)」→ 1그룹
📊 동사 활용표 — 정중형 · 과거 정중형
그룹기본형활용 방법정중형 (현재)과거 정중형
1그룹 会あう う단 → い단 + ます 会あいます 会あいました
行いく く → き + ます 行いきます 行いきました
よむ む → み + ます よみます よみました
はなす す → し + ます はなします はなしました
2그룹 見みる る를 빼고 + ます 見みます 見みました
食たべる 食たべます 食たべました
3그룹 来くる 불규칙 활용 来きます (き) 来きました (き)
する します しました
🔑 1그룹: 어미 う단 → い단으로 바꾸고 ます 붙이기
🔑 2그룹: 어미 る를 빼고 ます 붙이기
🔑 3그룹: 来くる → きます / する → します (암기 필수!)
🔄 현재 → 과거 바꿔보기 클릭하면 과거형 확인
まつりに 行いきます。→ きのう まつりに ___。
마쓰리에 갑니다. → 어제 마쓰리에 行いきました。
しんぶんを よみます。→ きのう しんぶんを ___。
신문을 읽습니다. → 어제 신문을 よみました。
えいがを 見みます。→ きのう えいがを ___。
영화를 봅니다. → 어제 영화를 見みました。
べんきょうを します。→ きのう べんきょうを ___。
공부를 합니다. → 어제 공부를 しました。
友ともだちと あそびます。→ きのう 友ともだちと ___。
친구와 놉니다. → 어제 친구와 あそびました。

동사 정중형: ～ます / ～ました
「~합니다 / ~했습니다」 — 동사의 정중형과 과거 정중형이에요.
현재·미래: ～ます   |   과거: ～ました
あした 友ともだちと 会あいます。

きのう 日本にほんの 友ともだちに 会あいました。
어제 일본 친구를 만났습니다.
夜よるの 東京とうきょうを 見みました。
밤의 도쿄를 봤습니다.
とても たのしかったです。
매우 즐거웠습니다.
きのう / あした / いま
시간 부사 — 언제인지를 나타내는 표현들이에요.
きのう = 어제 → ～ました (과거)   |   あした = 내일 → ～ます (현재·미래)
きのう まつりに 行いきました。
어제 마쓰리에 갔습니다.
あした えいがを 見みます。
내일 영화를 봅니다.
